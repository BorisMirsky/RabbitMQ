Как пользоваться


1
Завести виртуальное окружение. 
Запустить сервер Fast API: 'uvicorn producer:app --reload'       # 'producer'  - имя файла может отличаться


2
Запуск idle из под виртуального окружения 
Завести в. окружение
В терминале 'python -m idlelib.idle'    


3
hostname.py занесён в gitignore (т.е. пропущен т.к. содержит credentials)
В коде надо закомментировать строку 'from hostname import hostname' и вместо 
'hostname' использовать свою строку подключения вида:
"amqps://<login>:<password>@<server>.rmq.cloudamqp.com:<port>/<login>"
port = 5671 или 5672
